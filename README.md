## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

API-REST about cars and brands

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Use app

```bash
# add Data
localhost:3000/seed

# get cars
localhost:3000/cars

# get brands
localhost:3000/brands

```
